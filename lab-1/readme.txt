Вариант 32: Соединение слиянием упорядоченных списков (2)
При работе с базами данных часто приходится выполнять операцию соединения для двух и более таблиц.
Реализуйте алгоритм соединения для 2х таблиц слиянием упорядоченных списков. Исходные данные
храните во внешней памяти.

Для компиляции введите команду mingw32-make
Для запуска тестов - mingw32-make test

Исходный код хранится в папке src
Заголовочные файлы - в папке include
Тесты - в папке test

При компиляции программа оказывается в папке bin\join.exe
Она всегда запускается с параметрами.
Если параметр один, то это имя файла, который надо прочитать и вывести на экран.
Если параметров два, то первый из них имя файла, который надо сгенерировать, а второй - сколько в нём должно быть структур.
Если параметров три, то первые два из них - это имена файлов, которые необходимо слить, а третий - имя файла с результатом слияния.
